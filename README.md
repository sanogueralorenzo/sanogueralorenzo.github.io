## Intro

**Hello 👋** continuing with the spirit of this repo, this is an AI-tools-first orchestration playground.

## Quickstart

```shell
./install.sh
```

This installs:
- `codex-core`
- `codex-remote`
- `codex-menubar`

It does not install `overlay`, `voice`, or `site`.
