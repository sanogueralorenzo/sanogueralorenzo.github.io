# Unmade

Unmade is a scrollable archive of things that almost existed.

Unsent messages, unbuilt projects, unfinished films, abandoned maps, unrealized inventions, alternate lives, lost plans, and private ideas that never found a surface.

This is not a productivity site or a portfolio in the usual sense.
It is a public index of possible things: fragments, case files, prototypes, scenes, notes, and artifacts from work that was imagined but not completed.

## Sections

- **Unsent**: words that never reached anyone.
- **Unbuilt**: projects, tools, systems, and machines that stayed theoretical.
- **Unchosen**: paths, identities, and futures left behind.
- **Unfinished**: works abandoned mid-shape.
- **Unrecorded**: moments, places, and signals that almost disappeared.
- **Unlived**: versions of life that remained possible but unused.

## Shape

The site is meant to be browsed by scrolling, then entered through individual case pages.

Each entry should feel like an artifact: a name, a fragment, a visual trace, a short explanation, and the unresolved question it leaves behind.

## Local Preview

```shell
cd unmade
npm install
npm run dev
```

Then open `http://localhost:8080/unmade/`.
