# Unmade Build Goal

## Current Focus

Unmade is being moved into a standalone Eleventy site that lives entirely in `/unmade`.

The standalone skeleton is in place with its own package files, Eleventy config, source folder, layouts, styles, archive page, reusable case-page template, image asset path, and migrated entries. The archive has been refocused around three top-level buckets: Unsent, Unbuilt, and Unheld.

## Expansion Loop

Before adding new entries, keep upgrading the migrated entries in small quality-focused batches:

- Unsent: words and signals.
- Unbuilt: places, products, and software.
- Unheld: lives, relationships, roles, and futures.

Each batch should improve existing entries in place, add or wire at least one generated image asset when useful, keep the archive navigable, and run the narrowest relevant validation from `/unmade`.

## Next Expansion Focus

Continue improving the remaining entries within the three-bucket taxonomy. Do not restore deleted legacy buckets unless the top-level structure changes again.
