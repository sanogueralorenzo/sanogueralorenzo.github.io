The interface learned from rhythm: hesitation before a choice, the order of corrections, the pressure of returning to a page after midnight. It recognized people by the choreography of use rather than by names, passwords, or photographs.

The prototype felt kind because it never demanded an answer. It also felt worse than a demand because it assumed the answer had already leaked from the body.

It was rejected because recognition without naming felt both generous and invasive. The gentlest surveillance is still surveillance.

## Login Without Names

The interface is Unheld because the better user experience made the ethical problem harder to see. It removed the rude parts of identification while preserving identification itself.

The trick is not anonymity. The trick is whether a system can avoid names while still learning the shape of a person too well.
